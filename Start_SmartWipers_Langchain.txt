Start in virtual environment in SmartWiperAgents: (activate with "source .venv/bin/activate"):

	- Start Velocitas Runtime: velocitas exec runtime-local up

	- Run SmartWiper App: python -m app.SmartWiper

Does not need to be started in  virtual environment:

	- Start Ollama Server: ollama serve (optional: ollama pull llama3.1:8b)
